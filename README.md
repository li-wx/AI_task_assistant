An intelligent task management and scheduling assistant powered by Azure OpenAI that helps users organize, prioritize, and manage their tasks through natural language interaction.

## 🌟 Features

### Core Capabilities
- **Natural Language Task Management**: Create and manage tasks through conversational AI
- **Intelligent Task Decomposition**: Automatically break down complex tasks into smaller, actionable subtasks
- **Smart Prioritization**: AI-powered task prioritization based on urgency, importance, and deadlines
- **Dependency Management**: Identify and manage task dependencies automatically
- **Proactive Suggestions**: Get recommendations for related tasks you might have overlooked
- **Context-Aware Scheduling**: Uses current date/time and general knowledge for better assistance

### User Interface
- **Interactive Chat Interface**: Natural language conversation with the AI assistant
- **Task Management Dashboard**: Visual task management with filtering and sorting
- **Real-time Updates**: Live updates of task status and suggestions
- **Mobile-Responsive Design**: Works seamlessly on desktop and mobile devices

### Technical Features
- **Memory System**: Conversation history with intelligent summarization
- **Persistent Data Storage**: JSON-based storage for tasks and conversations
- **RESTful API**: Clean API for task operations and integrations
- **Azure Integration**: Built for Azure OpenAI and Azure App Service deployment
